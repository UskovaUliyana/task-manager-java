# Task Manager

Простая программа для управления задачами. Написана на Java, чтобы разобраться с массивами, циклами и логикой.

В программе можно добавить задачу, посмотреть список с номерами и статусами, отметить задачу как выполненную или удалить её. При удалении список автоматически сдвигается, пустых мест не остаётся.

Всё работает в консоли. Для ввода используется Scanner, задачи хранятся в массиве строк. 

Чтобы запустить: скомпилировать Quiz.java и запустить через java Quiz.
